# 4 C4 架构视图

> **AI提示**: 使用 PlantUML 绘制 C4 模型的三层视图。Context 展示系统与外部交互；Container 展示容器/服务架构；Component 展示内部组件结构。

## 4.1 Context View

> **AI提示**: 展示系统与外部角色/系统的交互关系。

```plantuml
@startuml
skinparam componentStyle rectangle
skinparam rectangle {
    BackgroundColor<<System>> LightBlue
    BackgroundColor<<ExternalSystem>> LightGray
}

actor "外部角色1" as ext1
rectangle "外部系统1" as extsys1 <<ExternalSystem>>
rectangle "MDC 系统" as mdc <<System>>

ext1 --> mdc : 交互描述
mdc --> extsys1 : 交互描述
@enduml
```

---

## 4.2 Container View

> **AI提示**: 展示 MDC 系统内部的容器/服务划分及交互。

```plantuml
@startuml
skinparam componentStyle rectangle
skinparam rectangle {
    BackgroundColor<<Container>> LightYellow
    BackgroundColor<<NewContainer>> LightGreen

rectangle "MDC 系统" <<System>> {
    rectangle "容器1" as c1 <<Container>>
    rectangle "容器2" as c2 <<Container>>

}

c1 --> c2 : 接口描述
c1 --> newc : 接口描述

@enduml
```
